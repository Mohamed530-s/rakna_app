# RAKNA | Premium Intelligent Parking 🚗

A high-end parking management solution built with **Flutter**, focusing on scalability, clean code, and "Cyberpunk Luxury" design.

---

## 🛠️ Tech Stack & Architecture
* **Clean Architecture**: Decoupled layers (Presentation, Domain, Data) for maximum maintainability.
* **State Management**: Advanced **BLoC & Cubit** logic for reactive and predictable data flow.
* **Patterns**: MVVM, Dependency Injection (GetIt), and Repository Pattern.
* **Backend**: Firebase (Auth, Firestore, Transactions) for real-time synchronization.
* **UI/UX**: Responsive & Adaptive design, ensuring a premium experience on both Mobile & Tablet.

---

## ✨ Core Features
* **Smart Booking**: Implements Atomic Firestore Transactions to prevent race conditions in spot availability.
* **Real-time Maps**: Custom cached markers with premium Dark & Silver Google Maps styling.
* **Dual Roles**: Integrated flows for both **Users** (Spot Booking) and **Guards** (QR Code Verification).
* **Dynamic Theming**: Cyberpunk Dark & Silver Light modes based on Material 3 standards.
